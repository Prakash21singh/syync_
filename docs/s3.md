### Package command
```
npm i @aws-sdk/client-s3
```

### Necessary imports
```
import {
  S3Client,
  PutObjectCommand,
  CreateBucketCommand,
  DeleteObjectCommand,
  DeleteBucketCommand,
  paginateListObjectsV2,
  GetObjectCommand,
} from "@aws-sdk/client-s3";
```
### Creating client
```
const client = new S3Client({
    region:process.env.AWS_REGION,
    credentials: {...} 
})
```

### Use cases. (exp)
```
const bucketName = `sync-cloud-storage`
await client.send(
    new CreateBucketCommand({
        Bucket:bucketName
    })
)
```
